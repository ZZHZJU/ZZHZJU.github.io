    张量

    在数学里，张量是一种几何实体，或者说广义上的“数量”。张量概念包括标量、向量和线性算子。张量可以用坐标系统来表达，记作标量的数组。标量可以看作零维张量，向量可以看作一维张量，矩阵可以看作二维张量，因此张量又称多维数组(multi dimension array) 

    import torch
import numpy as np

y = torch.rand(4,5)#产生一个四行五列的矩阵

    Functional 函数

    1. 激活函数：
    2. F.relu(input)：计算输入的 ReLU 激活函数，\(y=\begin{cases}0\qquad x\leq0\\x\qquad x>0 \end{cases}\)
    3. F.sigmoid(input)：计算输入的 Sigmoid 激活函数,\(y=\frac{1}{1+e^{-x}}\)，优点是求导容易且优化稳定，缺点是由于其软饱和性，容易产生梯度消失并且输出并不是以零为中心
    4. F.tanh(input)：计算输入的 Tanh 激活函数,\(y=\frac{1-e^{-2x}}{1+e^{-2x}}\)，优点是相比Sigmoid函数收敛更快且输出以零为中心，缺点依旧是没有改变由于饱和性产生的梯度消失

    5. F.softmax(input, dim)：计算输入的 Softmax 函数。

    6. 损失函数：

    7. F.cross_entropy(input, target)：计算交叉熵损失。
    8. F.mse_loss(input, target)：计算均方误差损失。

    9. F.binary_cross_entropy(input, target)：计算二分类交叉熵损失。

    10. 卷积操作：

    11. F.conv1d(input, weight, bias=None, stride=1, padding=0)：一维卷积操作。
    12. F.conv2d(input, weight, bias=None, stride=1, padding=0)：二维卷积操作。

    13. F.conv3d(input, weight, bias=None, stride=1, padding=0)：三维卷积操作。

    14. 池化操作：

    15. F.max_pool2d(input, kernel_size, stride=None, padding=0)：二维最大池化操作。

    16. F.avg_pool2d(input, kernel_size, stride=None, padding=0)：二维平均池化操作。

    17. 归一化操作：

    18. F.batch_norm(input, running_mean, running_var, weight=None, bias=None, training=False, momentum=0.1, eps=1e-5)：批量归一化。

    19. F.layer_norm(input, normalized_shape, weight=None, bias=None, eps=1e-5)：层归一化。

    20. 线性变换：

    21. F.linear(input, weight, bias=None)：执行线性变换操作，通常用于全连接层。

    22. 其他功能：

    23. F.dropout(input, p=0.5, training=True)：应用 Dropout 随机丢弃一些神经元，防止过拟合。
    24. F.interpolate(input, size=None, scale_factor=None, mode='nearest')：对输入进行插值操作（如上采样）。

    optim

    1. SGD（随机梯度下降）：
    2. torch.optim.SGD(params, lr=<学习率>, momentum=0, dampening=0, weight_decay=0, nesterov=False)

    3. 是最基础的优化算法，通过计算梯度来更新模型参数。可以配合 momentum（动量）来加速收敛。

    4. Adam（自适应矩估计）：

    5. torch.optim.Adam(params, lr=<学习率>, betas=(0.9, 0.999), eps=1e-8, weight_decay=0, amsgrad=False)

    6. 一种更复杂的优化算法，结合了动量和自适应学习率。通常在很多深度学习任务中表现优异。

    7. RMSprop（均方根传播）：

    8. torch.optim.RMSprop(params, lr=<学习率>, alpha=0.99, eps=1e-8, weight_decay=0, momentum=0, centered=False)

    9. 适用于非平稳目标函数，能够自动调整学习率。

    10. Adagrad（自适应梯度算法）：

    11. torch.optim.Adagrad(params, lr=<学习率>, lr_decay=0, weight_decay=0, eps=1e-10)

    12. 适用于稀疏数据集，能够自动调整每个参数的学习率。

    13. AdamW：

    14. torch.optim.AdamW(params, lr=<学习率>, betas=(0.9, 0.999), eps=1e-8, weight_decay=0.01, amsgrad=False)

    15. Adam 的一个变种，通过在更新过程中直接施加权重衰减，改进了正则化效果。

    16. Adadelta：

    17. torch.optim.Adadelta(params, lr=<学习率>, rho=0.9, eps=1e-6, weight_decay=0)

    18. 也是一种自适应学习率优化算法，适合于不同特征尺度的学习。

    19. LBFGS（有限内存 BFGS）：

    20. torch.optim.LBFGS(params, lr=<学习率>, max_iter=20, max_eval=None, tolerance_grad=1e-5, tolerance_change=1e-9, history_size=100, line_search_fn=None)
    21. 一种基于二阶导数的优化算法，适用于需要精确收敛的场景，通常用于较小规模的优化问题。

    3. torch.autograd.grad：

    4. 计算并返回输入张量的梯度。通常用于需要手动计算梯度的情况。

    1. torch.autograd.backward：
    2. 执行反向传播，用于计算损失相对于输入张量的梯度。

    1. torch.autograd.Function：
    2. 自定义求导方法。通过定义 forward 和 backward 方法，可以自定义前向和反向传播的行为。
    class MyFunction(torch.autograd.Function):
    @staticmethod
    def forward(ctx, input):
        ctx.save_for_backward(input)
        return input.clamp(min=0)

    @staticmethod
    def backward(ctx, grad_output):
        input, = ctx.saved_tensors
        grad_input = grad_output.clone()
        grad_input[input < 0] = 0
        return grad_input

    cuda

    以下是 torch.cuda 的一些常用功能：

    1. 检查是否有可用的 GPU: 

      torch.cuda.is_available()
      返回 True 如果有可用的 GPU，否则返回 False。

    2. 获取当前 GPU 设备的数量: 

      torch.cuda.device_count()
      返回可用的 GPU 设备数量。

    3. 设置当前 GPU 设备: 

      torch.cuda.set_device(device)
      设置当前默认的 GPU 设备，device 参数可以是设备的索引号（如 0、1）。

    4. 获取当前 GPU 设备: 

      torch.cuda.current_device()
      返回当前默认的 GPU 设备索引号。

    5. 将张量转移到 GPU: 

      tensor = tensor.to('cuda')  # 或 tensor.cuda()
      将 PyTorch 张量转移到 GPU 上以进行计算。

    6. GPU 设备上的内存管理: 

      torch.cuda.memory_allocated()
torch.cuda.memory_reserved()
      这些函数用于查询 GPU 上的内存使用情况。
