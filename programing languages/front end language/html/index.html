    HTML

    简介：

    HTML(HyperText Markup Language,超文本标记语言)是一种用于创建网站的标准标记语言。HTML可简单可复杂一切取决于web开发者。 HTML文件的扩展名是.htm或者.html

    Note

    1. HTML 标签不区分大小写，但最好用小写
    2. 空格回车不起作用
    3. 务必规范缩进

    HTML结构

    <html>
    <head>
    <!-头部内容->
    </head>
    <body>
    <!-网页内容->
    </body>
</html>

    HTML常用标签

    标签 属性 用途
    <html></html> 整个HTML文档
    <head></head> 文档头
    <tiltle></title> 文档标题
    <body></body> background 、bgcolor 文档主题内容
    <h1></h1>
    <h2></h2>
    ···
    <h6></h6>    		 标题
    <p></p>,<span></span> 段落，行内元素
    <br> 换行
    <pre></pre> 格式化文本
    <hr> 水平线
    <!-文字--> 注释文字
    a 超链接
    img 图片
    table 表格
    tr 表格中的一行
    th,td 表头/单元格
    ol,ul 有序/无序列表
    li 列表项
    dl 描述列表
    input name 用户输入域
    div 分区标签，组织网页结构
